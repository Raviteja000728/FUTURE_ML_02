# Support Ticket Classification System

The Support Ticket Classification System is a Machine Learning and Natural Language Processing (NLP) based web application developed using Streamlit. The system automatically analyzes customer support tickets, classifies them into relevant categories, assigns priority levels, and recommends the appropriate support team for resolution.

This project helps organizations streamline customer support operations by reducing manual ticket handling and improving response efficiency. The application processes user-entered ticket information and provides intelligent recommendations based on the ticket content.

## Features

* Customer Ticket Submission Interface
* Automatic Ticket Category Classification
* Priority Prediction (High, Medium, Low)
* Support Team Assignment
* Service Level Agreement (SLA) Recommendations
* Interactive Streamlit Dashboard
* NLP-based Text Processing
* Real-time Ticket Analysis

## Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Naive Bayes Classification
* Natural Language Processing (NLP)

## Project Objective

To automate the classification and prioritization of customer support tickets, enabling faster issue resolution, improved customer satisfaction, and efficient support team management.

## Expected Outcomes

* Reduced manual ticket sorting effort
* Faster response times
* Better ticket prioritization
* Improved support workflow efficiency
* Enhanced customer service operations
